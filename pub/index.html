Telegram Send Message Bot

Простой способ отправлять сообщения в Telegram через HTTP API

Как начать использовать

  1. Добавьте бота в Telegram: @tgsendmsgbot
  2. Отправьте боту команду /start
  3. Бот пришлёт вам уникальный токен для API
  4. Используйте токен в GET-запросе по шаблону

Пример API запроса

GET https://tgsm.bot.nu/api/{token}/{message}

Где token — ваш токен из бота, message — текст сообщения (URL-кодированный)
